## What it does
The following features available today.

### Skillbook
The skill book is a powerful tool for buildcrafters, or when you just want to find that one skill to equip. It allows filtering and sorting of all skills in the game.

For example it could be used to:
* Find all stance skills
* Find all skills giving health regen
* Find all skills related to knock-down
* Find all skills giving +damage and sort by amount

Or even, if you've completely lost your mind:
* Find all hexes, in prophecies or nightfall, yet to be unlocked, having a recharge of 10 seconds or less. Sort by energy cost, then by AoE range.

## FAQ
- ### Help! My antivirus software is flagging ***Hero‚Äôs Insight*** as Malware. Have I been infected?
  If you have downloaded the mod from this Github repository: No you have not been infected. You can safely add **Launch_HerosInsight.exe** to you antivirus software's whitelist/exlusion list and keep using the mod.

  Explanation: Antivirus software uses a bunch of heuristics to determine if something is malware or not. Sometimes these heuristics fail and you get false positives. In the case of ***Hero‚Äôs Insight***, the antivirus software maybe saw that its a lesser known exe, maybe few users etc. Maybe it checked the code and saw that it uses dll injection, a method also used by malware sometimes, however in this case for a completely benign purpose.
